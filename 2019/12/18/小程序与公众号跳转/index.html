小程序与公众号跳转

发表于 2019-12-18
字数统计: | 阅读时长 ≈

##小程序跳转公众号
1.在小程序中使用webview
1.1.webview 指向网页的链接。可打开关联的公众号的文章，其它网页需登录小程序管理后台配置业务域名
1.2.可以增加公众号的阅读量，不能点赞、收藏等
1.3.地址：https://developers.weixin.qq.com/miniprogram/dev/component/web-view.html
2.使用official-account组件关注公众号
2.1.进入公众号关注页面，设置的公众号需与小程序主体一致，宽高限制w300*h84，每个页面只能配置一个该组件
2.2.https://developers.weixin.qq.com/miniprogram/dev/component/official-account.html
3.小程序客服会话框回复，返回公号文章
3.1.小程序配置客服会话按钮，点击进入客服:
3.1.1.按钮配置：show-message-card=“true”
3.1.2.地址：https://developers.weixin.qq.com/miniprogram/dev/component/button.html
3.2.用户进入客服页面，回复相关消息(跟文章相关)
3.3.接收用户消息：https://developers.weixin.qq.com/doc/offiaccount/Message_Management/Receiving_standard_messages.html
3.4.分析用户消息并发送消息：https://developers.weixin.qq.com/doc/offiaccount/Message_Management/Passive_user_reply_message.html

###扩展：
1.小程序间跳转：
1.1.需要用户触发跳转
1.2.需要用户确认跳转
1.3.源小程序与目标小程序不再需要绑定至同一个公众号
1.4.每个小程序可跳转的其他小程序数量限制为不超过10个
地址：https://developers.weixin.qq.com/community/develop/doc/0000ac39168d50641477b725c51c01

2.微信公众号，点击菜单，跳转到小程序：可以到具体页面
参考地址：https://www.cnblogs.com/quan-coder/p/9158753.html
2.1.可带参数：onLoad: function (options) {}
